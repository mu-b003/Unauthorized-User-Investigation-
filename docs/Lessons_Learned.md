# Lessons Learned

## Technical Lessons

- User creation events leave valuable forensic evidence.
- journalctl provides a complete audit trail.
- Sudo assignments should be monitored continuously.
- Unauthorized accounts are easily identified through user enumeration.

---

## SOC Lessons

- Always verify privileged accounts.
- Build an attack timeline before drawing conclusions.
- Collect evidence before making changes.
- Preserve logs whenever possible.
